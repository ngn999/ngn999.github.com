2.2 mode line

When you are using a character-only terminal (not a window system),
CS uses three characters to describe, respectively, the coding system
for keyboard input, the coding system for terminal output, and the
coding system used for the file you are editing.


3 useful shortcut    shortcuts

3.2 emacs

3.2.1 org

  • C-c @
    select entry to export

3.2.2 register

  • C-x r w <Register>
    save frame state to Register
  • C-x r Spc <Register>
    save current postion to Register
  • Cx r j <Register>
    jump back

3.2.3 C++

  • C-M-e , C-M-a,
    goto end/start of current function,
  • C-c C-q
    indent current function
  • C-M-h
    select whole function
  • M-;
    comment current line
