Using Bootstrap grid

  •  Adding icons
  • 测试页面 

Bootstrap scaffolding could be defined using both #+begin_o_blog_row and #+end_o_blog_row directives (or <og TAB shortcut). A new column could be started using #+o_blog_row_column single directive (or <ogr TAB shortcut).

Example 

#+begin_o_blog_row 2 -1 show-grid
Column 1
#+o_blog_row_column 2
Column 2
#+end_o_blog_row

Output

Column 1

Column 2

The syntax is: 

#+begin_o_blog_row SPAN OFFSET GRID-CLASS
Column 1
#+o_blog_row_column SPAN OFFSET
Column 2
...
#+o_blog_row_column SPAN OFFSET
Column n
#+end_o_blog_row
